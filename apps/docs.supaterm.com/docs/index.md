Supaterm is an agent-first native macOS terminal built with libghostty. It combines a fast terminal with spaces, tabs, panes, coding-agent awareness, and the `sp` command-line interface.

## Start here

1. [Install Supaterm](/guides/get-started/install).
2. [Take the first-launch tour](/guides/get-started/first-launch).
3. [Organize a project in its own space](/guides/get-started/first-space).
4. [Connect your first coding agent](/guides/get-started/first-agent).
5. [Automate Supaterm with the CLI](/guides/cli).

Already comfortable in the app? Open the [CLI quickstart](/guides/cli) or jump to [troubleshooting](/guides/troubleshooting).
