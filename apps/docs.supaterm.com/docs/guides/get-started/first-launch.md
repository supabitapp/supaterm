On a fresh installation, Supaterm opens a terminal and runs `sp onboard`. You can run it again at any time:

```bash
sp onboard
```

## Learn the layout

Supaterm has four organizational levels:

- **Spaces** separate areas of work.
- **Groups** optionally collect related tabs inside a space.
- **Tabs** live at a space root or inside a group.
- **Panes** are split terminal regions inside a tab.

The sidebar shows spaces, root tabs, colored groups, pinned work, unread activity, working directories, and coding-agent status. Toggle it with `Command-S`.

## Essential shortcuts

| Action             | Shortcut           |
| ------------------ | ------------------ |
| Command palette    | `Command-Shift-P`  |
| New tab            | `Command-T`        |
| Split right        | `Command-D`        |
| Split down         | `Command-Shift-D`  |
| Find in terminal   | `Command-F`        |
| Toggle sidebar     | `Command-S`        |
| Toggle agent panel | `Command-I`        |
| Close pane         | `Command-W`        |
| Close tab          | `Command-Option-W` |

Open the command palette when you do not remember a shortcut. It searches Supaterm actions, Ghostty actions, spaces, tabs, and panes.
