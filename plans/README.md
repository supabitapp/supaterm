## Findings considered and rejected

Recorded so they aren't re-audited. All were vetted against the code at `5663d0d`:

- **"Committed credentials in `.env`"** — false: untracked, gitignored, never in git history, mode 0600. Conventional local signing-secrets file. No action (rotation unnecessary).
- **Socket fd leak in `SocketControlRuntime.writeResponse`** — false: the early `return` exits the `withUnsafeBytes` closure; `close()` always runs.
- **AsyncStream continuation-swap race in `requests()`** — false: actor-isolated synchronous code, no suspension point in the window.
- **`AgentTranscriptTailer` losing partial JSONL lines** — false: the cursor only advances past complete lines; truncation triggers `didReset`.
- **Socket permission race at bind time** — mitigated by design: directory enforced 0700/owner-checked before bind; chmod 0600 before `listen`.
- **"No auth on the socket RPC"** — by design: same-user UDS trust boundary (0700 dir), same model as tmux/ssh-agent.
- **Tag injection in `bump_and_release.py`** — false: `subprocess.run` with list args throughout, no `shell=True`.
- **"CI rebuilds Ghostty every run"** — false: `.github/actions/setup-macos` caches ghostty/zmx/Tuist keyed by submodule SHA.
- **"Codex monitor re-decodes the whole transcript per poll"** — false: tailing is incremental; decoding is per new record.
- **SwiftUI whole-state observation re-render storm** — unproven: `@Observable` tracks per-property access; needs Instruments evidence before any plan.
- **PostHog v2 / TanStack Router v2 migrations** — nonexistent versions; both libraries are current at v1.
- **Worker meta-rewrite ReDoS** — the `[^"]*` pattern is linear; not exploitable.
- **`test.yml` calling `mac-test-xcodebuild` instead of `mac-test`** — intentional CI step-splitting for caching/visibility.
- **Socket protocol version field** — rejected: the app injects its bundled CLI (`SUPATERM_CLI_PATH`) into panes, so version skew isn't a real deployment mode; forward-only.
- **O(n) `firstIndex` scans in TerminalHostState** — real pattern, desktop scale (tens of tabs); not worth doing.
- **Three-installer duplication (Claude/Codex/Pi)** — weaker than reported: the shared core (`AgentHookSettingsFileInstaller`) already exists; Pi legitimately differs. Not worth a generic abstraction now.
- **Hook-install read-modify-write lost-update** — real but rare (finding #8 in the audit); deferred by maintainer selection. Revisit if users report clobbered agent settings.
- **`test.yml` missing concurrency cancellation; release actions pinned by tag not SHA; hero.png 1.5 MB; 300s download cache** — confirmed-but-unselected smalls (findings #6, #10, #11); fair game for a future quick pass.
